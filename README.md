## 需求
* 讓懶得手寫簡譜的人可以用點擊完成簡譜
* 讓沒有音樂知識的人也可以透過顏色看得懂譜（就是我）
* ~~基本上是為了使用完美鋼琴APP而客製化的工具~~

## 開發工具
* jQuery
* Javascript
* HTML
* CSS

## 功能介紹
* 鋼琴面板：下方鋼琴面板。
    * 新增音符：按下鋼琴面板中的琴鍵，即可於畫面上產生對應的音符。
* 樂譜面板：顯示音符的地方。
    * 修改歌名：雙擊歌名區塊可以輸入歌名。
    * 修改軌道顏色：雙擊軌道區塊可以選擇顏色。
    * 刪除指定軌道：點擊軌道右上方的叉叉按鈕，即可刪除該軌道及內容。
* 工具箱面板：右方工具箱（開啟；收合）。
    * 刪除：刪除一個音符或軌道。
    * 空格：增加空格於音符間。
    * 新增一行：增加新的一行軌道。
    * 關閉/開啟鋼琴：可控制鋼琴面板的顯示（開啟；收合）。
    * 切換模式：可控制樂譜面板的顯示模式（簡譜；五線譜）。
